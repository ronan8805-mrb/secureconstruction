## 🆘 Common Questions

### "How do I edit the website?"
Open the HTML files in any text editor (Notepad++, VS Code, or even Notepad). Find the text you want to change and edit it. Save and re-upload to your hosting.

### "How do I add more project photos later?"
Open `projects.html`, find a project section, and replace the placeholder image with your new photo. Follow the same HTML structure.

### "The mobile menu isn't working?"
Make sure you uploaded the `js/script.js` file to your hosting. Check that the file path is correct.

### "How do I know if people are visiting?"
Install Google Analytics (instructions in SEO-GUIDE.md). You'll see visitor numbers, pages viewed, and where they came from.

### "Can I change the colors?"
Yes! Open `css/styles.css` and look at the top section with color variables. Change the hex codes to your preferred colors.

### "How do I add a blog later?"
Create a new file called `blog.html`, add it to the navigation menu, and follow the same structure as other pages. Or wait for Phase 2 (see README.md).
